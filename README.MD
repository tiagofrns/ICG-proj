# Padaria Artesanal

Jogo de gestão de padaria em 3D feito com Three.js, inspirado no Overcooked.

## Progresso

- [x] Setup Three.js — renderer, cena, câmara, luzes
- [x] Cenário completo — paredes, balcão, forno, prateleiras e decoração
- [x] Jogador com movimento (WASD) e animação de caminhada
- [x] Mecânica do pão — pegar, amassar, enfornar, tirar
- [ ] Clientes com pedidos e barra de paciência
- [x] Sistema de dinheiro e HUD
- [ ] Receitas diferentes por produto
- [ ] Progressão de dificuldade
- [ ] Ecrã de fim de jogo
- [ ] Sons e música
- [ ] Colocar num servidor online (github?)
## Como correr

Abre o `index.html` no browser.
